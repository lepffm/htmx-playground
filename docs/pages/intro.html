    htmx is light-weight JavaScript library for performing AJAX requests, triggering CSS transitions.

    features
    - lightwight - 157 kB (min.js 46.7 kB) cf) react
    - less javascript (JS-in-HTML), use html extended attribute, aka hyper-script
    - supprot GET,POST + PUT, DELETE, PATCH, websocket
         <form hx-post="/submit-form" hx-swap="outerHTML" hx-target=#formResponse> 
    - support client template for json
    - support trigger  hx-trigger="every 10s"     
    - github accelerator
